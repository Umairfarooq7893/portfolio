Go to GitHub
Click New Repository
Name it:
👉 portfolio
Click Create Repository


👉 Method 1 (Simple Upload)
Open your repo
Click Add file → Upload files
Upload:
index.html
style.css
script.js
Click Commit change


👉 Method 2 (Using Git — better for you as IT student)

Open terminal in your project folder:

git init
git add .
git commit -m "initial portfolio"
git branch -M main
git remote add origin https://github.com/your-username/portfolio.git
git push -u origin main


Go to Repo Settings
Scroll to Pages
Under Source:
Select → main branch
Folder → /root
Click Save
